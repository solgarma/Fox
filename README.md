https://github.com/mgrmsc/lusrmgr/releases/tag/1.6.3

Local User and Group Management acts as an alternative to the native lusrmgr snap-in, bringing extended and more advanced tools for user and group administration to every Windows edition.

## System requirements

* Requires .NET Framework version 4.6 or higher
* Minimum supported operating systems: Windows Vista or Windows Server 2008

## Features

* Delivers improved utilities for handling users and groups
* Fully supports existing security principals
* Comes with a built-in search feature
* Allows remote connections for managing users and groups on external systems
* Offers a user interface that can be customized to personal preferences
